1. What is Network?
A group of computers connected together so they can share files and other resources between them.
Eg:

Computer A
Computer B
Computer C

All are connected through the same network.

2. What is an IP Address?
IP Address is a unique address of your machine on network. It also helps to find another computer on network.

3. What is Shared Folder?
A shared folder is a folder that other computers on network are allowed to access.
Eg:

On a Windows computer:
D:\SharedData

It is shared on the network as:
SharedData

Other computers can access it.

4. What is Mount Point?
A mount point is a directory where another filesystem is attached.

Example:
sudo mkdir /mnt/shared

After mounting, the files from the remote computer appear inside:
/mnt/shared

5. What is Mounting?
Mounting means connecting storage device or shared folder in a network so that you can access the files at other end.

Example:

Windows Shared Folder
        │
        ▼
Linux Mount Point
/mnt/shared

After mounting, you can use commands like:

cd /mnt/shared
ls
cp
mv

just like a normal folder.

6. What is SMB?
SMB (Server Message Block) is a protocol to share files and printers over network.
CIFS - It is a version of SMB protocol.

What is cifs-utils?
It is a linux package that allow linux to connect Window shared folders.

Install it using:
sudo yum install cifs-utils -y

or

sudo apt install cifs-utils -y

CIFS (Common Internet File System)
i) Mount Windows shared folder
Syntax: sudo mount -t cifs //server-ip/share-name /mount-point -o username=<username>

Eg: sudo mount -t cifs //192.168.1.10/SharedData /mnt/shared -o username=john

Explanation
mount → Mount a filesystem.
-t cifs → Specify the filesystem type as CIFS.
//192.168.1.10/SharedData → Windows shared folder.
/mnt/shared → Local directory where the share is mounted.
username=john → Windows username used to access the shared folder.

After running the command, the shared files are available in:

/mnt/shared

ii) Check Mounted file System by running "mount" or "df -h" command
 - Defines whether filesystem is mounted or not.

iii) Umount the share
umount /mnt/shared
 - Disconnects the shared folder from linux system.

iv) Automatically mount at boot
Add an entry in /etc/fstab

//192.168.1.10/SharedData /mnt/shared cifs username=john,password=your_password 0 0
Note:- Don't store your password in fstab file.
