Shared Folder

To install Samba,
a) RHEL - package-name: samba-client
b) Ubuntu - package-name: samba

Create a Shared Folder
sudo mkdir /shared

Give permissions:
sudo chmod 777 /shared

Edit Samba Configuration file
/etc/samba/smb.conf

Add the following at the end:

[Shared]
path = /shared
browseable = yes
writable = yes
guest ok = yes
read only = no

Explanation
Shared → Name of the shared folder.
path → Location of the folder.
browseable = yes → Users can see the shared folder.
writable = yes → Users can create, edit, and delete files.
guest ok = yes → Allows access without a username and password.
read only = no → Users can modify files.

Start Smba service
sudo systemctl start smb

Enable Samba at boot time
sudo systemctl enable smb

Check Samba Status
sudo systemctl status smb

Create a Linux user if it doesn't exist
sudo sueradd john 

Create samba password for john
sudo smbpasswd -a john

Check samba configuration for error. Run where Smba is installed
testparam

****Access the Shared Folder****
From Windows

Open Run (Windows + R) and type:

\\192.168.1.10\Shared

Replace 192.168.1.10 with the IP address of your Linux server.

Note:- configuration file define what is the name of network and which folder you are sharing.
