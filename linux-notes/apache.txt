Apache is a software that help users to display webiste over internet or on a local network.
It receive request from user (web browser) and send web pages back to them.
Eg:
 - user enter http://example.com in web browser
 - The browser send request to apache server
 - Apache find requested web page and send back it to client.

Why it is used?
 - Serve website and web application.
 - Store HTML, CSS, Javscript and other files on it's server.
 - Supprt PHP and other server side languages.
 - Hanle multiiple users at same time.

To install Apache,
 - Ubuntu:
      sudo apt install apache2 -y
 - RHEL:
      sudo yum install httpd -y

Start Apache
sudo systemctl start httpd

Starts the Apache service.

On Ubuntu, use apache2 instead of httpd.

Stop Apache
 - sudo systemctl stop httpd
Stops the Apache service.

Restart Apache
 - sudo systemctl restart httpd
Restarts the Apache service.

Reload Apache
 - sudo systemctl reload httpd
Reloads the configuration without stopping the service.

Enable Apache
 - sudo systemctl enable httpd
Starts Apache automatically after every system boot.

Disable Apache
 - sudo systemctl disable httpd
Stop Apache from starting automatically after boot.

Check Apache Status
 - sudo systemctl status httpd
Check if service is running or not

Check Apache Configuration
 - sudo apachectl configtest

Output:
Syntax OK

That means Apache is running fine

Check Apache version
 - httpd -y

It will palce website files in /var/www/html location.

Eg:

Create a simple web page
echo "<h1>Welcome to My Website</h1>" > index.html

For Ubuntu, the main configuartion file for Apache is /etc/apache/apache2.conf
For RHEL, it will be /etc/httpd/conf/http.conf

It store the information about user request
/var/log/httpd/access_log

It store logs about error
/var/log/http/error_log

Allow user to access the website over HTTP
sudo firewall-cmd --permanent --add-service=http
sudo firewall-cmd --reload

You can add service for HTTPS whhich contains port no 443.
Shows whether Apache is running or stopped.
