sudo shutdown -h now

1. pwd ==> Print Working Directory
Shows on which path you are in
Command: pwd

Output:
/home/bilal/Documents
This means you are currently inside Documents directory.

2. ls ==> List
Display files and folders in current directory
Command: ls

Output:
file1.txt
notes.txt
Documents
Pictures

3. ls -l
Display files and folders with detailed list such as permission, owner, size and date.
Command: ls -l

4. ls -a
Display files including hidden files (starting from .)
Command: ls -a

5. cd (Change Directory)
Move from one directory to another
Command: cd Documents
Moves into Documnets directory.

Go back one directory
cd ..

Go to the home directory
cd

6. mkdir (Make directory)
Creates a new directory/folder
Command: mkdir project

Creates a directory called project.

7. rmdir (Remove Directory)
The directory must be empty, then only it remove directory
Command: rmdir project

8. touch
Creates an empty file
Command: touch notes.txt

9. rm (Remove)
Command: rm filename

If you run rm -r <directory-name> then it will delete a directory and everything inside it

10. cp (Copy)
Command: cp notes.txt backup.txt
It create a copy of notes.txt called backup.txt

11. mv (Move)
Moves a file to another location or renames it.

i)Rename a file:
mv notes.txt mynotes.txt

ii)Move a file:
mv notes.txt Documents/

12. cat
Displays the contents of a file.

Command:
cat notes.txt

Example output:
Linux is an operating system.

13. head
Displays the first 10 lines of a file.

Command: head notes.txt

14. tail
Displays the last 10 lines of a file.

Command: tail notes.txt

15. echo
Displays text on the terminal or writes text to a file.

i)Display text:
echo "Hello"

Output:
Hello

ii)Write text to a file:
echo "Hello Linux" > file.txt

ii)Append text to a file using >> symbol

16. whoami
Display the username of currently logegd in user.

17. w
Who is currently loggedin and what they are doing

Command: w

Ouput:
USER     TTY      FROM         LOGIN@   IDLE   JCPU   PCPU   WHAT
bilal    pts/0    192.168.1.5  10:30    2:00   0.10s  0.01s  vim notes.txt

Explaination:
 - USER → Logged-in username
 - TTY → Terminal being used
 - FROM → IP address or login location
 - LOGIN@ → Login time
 - IDLE → How long the user has been inactive
 - WHAT → Current command or program the user is running

Press Ctrl + C to stop.

24. telnet
Checks whether remote server or port is reachable or not.
It commonly used to test network connectivity to verify whether receiver can able to accept connection.

Syntax: telnet <ip or hostname> <port-no>

Eg:
i)Test HTTP (Port 80)
Command: telnet google.com 80

If connection is successful then it will give output

Trying 142.250.xx.xx...
Connected to google.com.
Escape character is '^]'.

which means port 80 is open and reachable

ii)Test SSH (Port 22)
telnet 192.168.1.10 22

If it connects, the SSH port is open.

If connection fails, then it will give output

Trying 192.168.1.10...
telnet: Unable to connect to remote host: Connection refused

That means service is not running or port is blocked by a firewall

Note:- Telnet port no is 23
