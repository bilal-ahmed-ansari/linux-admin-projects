tar:
Earlier store backup in tape, hence it is called tape archive. Archive means group file and folder together in one package (file)
Tar is used to convert multiple files and folder together in one single archive file.

/home/bilal/project

file1.txt
file2.txt
file3.txt
images/
logs/
tar -cvf project.tar /home/bilal/project
-c = create new package/archive
-v = while doing archive, you will see files on the screen
-f = specify name of the file
-t = to see which files are present in archive
-x = to extract the files
Output: project.tar
Note - tar file will not compress
Tar is used for archiving the logs, taking backup

Zip archive and compress, used in windows

Tar + compression
1.gzip (.tar.gz)
tar -czvf backup.tar.gz /home/bilal/backup
z = gzip compression
