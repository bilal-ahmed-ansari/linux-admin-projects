Linux file system:
/ is a root directory which contains an entire os
Top level directory 
/Home
You will see the personal folder of user
/Root
This is personal directory of root user
/Etc 
You will see the configuration files
/Var
Type of data is variable which means data is changing continuously 
/Tmp - temporary files
Sometimes application store data temporary 
All the files will be deleted after reboot
/boot
Required files for reboot 
Kernel files, grub, initramfs
Eg: /boot/vmlinuz - kernel
/Dev
Device are stored in form of file
Example

/dev/sda
Hard Disk
/dev/sdb
USB
/dev/null
Special device
/dev/tty
Terminal

/Proc
It gives system live information 
It is like a virtual file system
Example

cat /proc/cpuinfo
CPU details
cat /proc/meminfo
RAM details

/Sys
It provides kernel and hardware information 

/Bin
/Bin/ls will actually run /usr/bin/ls

Bin - system basic command
Eg
/bin/ls
/bin/cp
/bin/mv
/bin/cat
/bin/mkdir
/bin/rm

/Sbin
These command used to manage system, mostly used by administrator 
/sbin/fsck
/sbin/reboot
/sbin/shutdown
/sbin/mkfs

/Usr
Here we store application, command, libraries and documentation

/opt
Third-party software.

Example

/opt/google
/opt/zoom

/Lib
Windows having dll files, same way linux contain libraries

/Mnt
Used for temporary mount point
Mount /dev/sda /mount

/Media
When you attach usb, cd/dvd drive, it automatically mount

/Srv
All type of server data are stored here
