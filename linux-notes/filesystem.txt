Filesystem: 
Filesystem decide how to store the data, where to store the data, how we can find the data.
Means it organize and group the data if required.
1. Nfs
Network file system 
From one server we can access the files of another server
2.ntfs
Mostly used in windows
3.swap
If ram is full then some part of disk will acts as as memory
Ram full then swap memory usage
4.xfs
Used in Rhel, centos, rocky linux
Support large group of data
It will grow but it won't shrink
5.ext4
Common in ubuntu
It will grow and shrink
It is very fast and stable.
6.iso9660
Mostly used for Cd/dvd images.
7.zfs
Zettabyte file system
If there is data in hard disk and entite hard disk or data get corrupt then we will lost our data.
It will check the data automatically, if disk is corrupt then it will detect it. Compress the data and support for larger storage.
8. Btrfs
B tree file system
For complex Raid setup, this filesystem is required.
Support compression, large storage of data and provide data protection.
Also it is a modern file system.
