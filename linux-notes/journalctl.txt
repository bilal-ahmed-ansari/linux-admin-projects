consider linux is a office. We have recorded each event in a diary.
Eg
user has login
ssh has failed
apache has crashed
nginx has started
server has rebooted
service has stopped

Command:
journalctl
journalctl -e = give latest logs
journalctl -f = live logs
journalctl -u ssh = ssh logs  [rhel]
journalctl -u ssh.service [ubuntu]
journalctl -b = give all logs after current boot
jornalctl -b -1 = give logs of last reboot
jornalctl --today = today log
journalctl --since "1 hour ago" = before 1 hour log
journalctl -p err = error logs
