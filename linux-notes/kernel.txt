It is a core of os. It acts as a bridge between application (user) and cpu, ram, hard drive, keyboard (hardware)
Eg when you run ls command, ls won't directly communicate with hard disk. It send request to kernel, kernel read the data from disk and pass back to ls

Component of kernel
a.Process management 
Manage the process by creating and terminating the process, allocating cpu, cpu scheduling 
Command 
ps,top,htop,kill
Kernel decide which process will get priority to use cpu and ram
b. Memory management 
Manages the Ram by create and allocate memory, free some memory, use swap memory if possible, protect memory
Command
vmstat
If ram is full then it kill out of memory (oom)
c. Disk management 
Manage files which are present in hard disk
It will read, write, delete the data from disk
Manage permission 
Command
mount 
df -h
ls
cp
mv
Kernel will understand the filesystem like ext,xfs
d. Device driver
Control hardware like cpu, ram, mouse, keyboard 
e.Network slack
Handle network communication 
Eg
Ssh
Tcp/ip
Ftp
Http
f. Security
Kernel maintain security like selinux, hardening
