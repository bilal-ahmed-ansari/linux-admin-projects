When linux machine turn On, bios/uefi start intializing hardware devices like cpu, ram, keyboard.
It find bootable device like ssd, hdd, usb

Always bootloader is known as grub
It load linux kernel into memory and grant control to kernel

Kernel manage cpu scheduling, memory management, loading device driver, mounting root filesystem

Initramfs (initial ram filesystem)
Kernel uses intiitrams which contains temporary drivers and scripts to locate root filesystem and mount it.

Then kernel start user space process which called systemd having pid 1

Then systemd start background service like ssh, networking, logging, database

Then user can get login screen and he or she can able to login
