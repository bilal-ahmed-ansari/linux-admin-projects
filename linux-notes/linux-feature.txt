Feature of linux
a. Open source
Source code is freely available. Anyone can see and modify the code, create his own version of linux
b. Multiuser
Multiple user login on same linux machine
c. Multitasking
At a time multiple program is running
d. Security
In linux we are using security features such as file permissions, selinux /apparmor, firewall
e Stable
Server will be stable even if we don't reboot in months if there is no update in kernel 
f. Portable
Linux will run on every hardware like desktop, laptop, cloud, server
g. Command line tool is powerful in linux. It's because using this command we can run automation
10. Linux work on less resources which results high performance
