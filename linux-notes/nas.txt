network attached storage
it is a storage device which is connected to our network and share the files.
consider there is a folder in office
\\NAS-Server\Projects
every users can able to access this folder

Diagram:
        PC1
          \
        PC2 ----> NAS ----> Hard Disks
          /
        PC3
Here we are using LAN cables to acess the folder ans store in NAS which is attached in hard disk.
Protocols:

NFS (Linux)
SMB/CIFS (Windows)

mount on linux

sudo mount -t nfs 192.168.1.10:/data /mnt/nas

Or SMB:

sudo mount -t cifs //192.168.1.10/share /mnt/nas
