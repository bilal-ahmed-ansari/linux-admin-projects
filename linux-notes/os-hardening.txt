Secure the server by removing unnecessary things so that the attacker won't attack our system.
While installing linux secure system as much as you can

Step 1: update the system
apt update
Go to repository, check the version of package. If there is any new package, update its index and download that package in our machine
For eg if we have chrome 136 version and in repo, it is showing 137 version then it tells that 137 version is available for chrome.

apt upgrade - update your package actually in your system
Step 2
Disable unnecessary services

Note: ftp service name is vsftpd

Step 3 
Configure firewall 
Ubuntu - ufw enable
Rhel - firewall-cmd

Step 4
Disable ssh root login
Go to /etc/ssh/sshd_config
PermitRootLogin No

Keep Strong password, use ssh keys, remove unnecessary users
Don't provide 777 permission to file
