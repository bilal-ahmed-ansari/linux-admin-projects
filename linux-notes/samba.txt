Samba:
It is a software which allow linux and windows to share files and printer between them
Step 1
Consider there is a folder called /shared in linux server
Step 2
User open file explorer on windows and type
\\linux-server-ip\shared
Step 3
Linux and windows don't know each other language. Windows tell I want to use shared folder which is present in linux server. To communicate to linux server windows using smb protocol so that linux can understand 
Step 4
Linux server (smbd) will receive the request and check if that folder really exist in its machine, if it is allowed to share, check whether any client is an authorised person and if server has permission to send data to client 
Step 5
We need to write in samba configuration file
[shared]
path = /shared
Step 6
Now windows can access the file
