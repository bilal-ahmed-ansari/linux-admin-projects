Sar - system activity reporter
Usually used to check server performance 
It will show current and past performance of your system 
Ubuntu package - sysstat
When you run sar, it shows the statistics of cpu
12:00:01 AM %user %system %iowait %idle
12:10:01 AM 5 3 1 91
1st field - how much application used by cpu
2nd field - kernel use system 
3rd field - Cpu wait for disk. If it high, then cpu will slow
4th field - if it is 95% then cpu is free 
For cpu, use sar -u
For ram use sar -r = it show whether ram is using or not
sar -d = it will show disk read and write activity 
sar -S = it will check whether swap is using or not
sar -u 5 3 = check cpu every 6 sec, 3 times. Also it shows the average 
sar -u -f /var/log/sysstat/sa17 = show data for 17th date
