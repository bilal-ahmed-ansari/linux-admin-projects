It provides security in linux server
It decide which process have permission to access the file and which are not.
Provides an extra security layer.
Having 3 modes:-
a)Enforcing - we need to create the rules
b)Permissive - rules will break but we record the logs of those one.
c)Disabled - SELinux is OFF.
getenforce ==== show permission details
setenforce 1 ===== make permission as an Enforcing.
setenforce 0 ===== make permission as Permissive.
