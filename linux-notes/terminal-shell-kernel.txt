Terminal:
It is an interface where you can enter commands and see the output
Eg: gnome terminal, konsole,xterm

Shell:
It is a command interpreter where it will read those command and move that command to os and will show an output 
Eg: bash, sh, zsh,ksh, fish
When you type ls command, terminal give to shell, shell run ls program and give output
